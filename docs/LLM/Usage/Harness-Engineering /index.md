# Harness Engineering

## openai Ryan Lopopolo

Ryan Lopopolo is a software engineer at OpenAI focused on new ways of building software with agents. He writes and speaks about Harness Engineering: the systems, constraints, and **feedback loops** that make coding agents useful in practice.

**Coding agents** are getting good enough that “can the model write code?” is no longer the most interesting question. The more important question is: what kind of **engineering system** do you need around the model to make its work useful, repeatable, and trustworthy?
